# GeoG — Fiches store localisées (ES / PT-BR / DE / IT)

Objectif ASO : ouvrir 4 marchés énormes pour un jeu de géo mondial. À coller dans
**Google Play** (Croissance → Présence Play Store → gérer les traductions) et
**App Store Connect** (chaque localisation).

Notes importantes :
- **Google Play n'a PAS de champ mots-clés** → les mots-clés doivent être tissés
  dans le titre + la description courte + la description longue (déjà fait ci-dessous).
- **App Store a un champ « mots-clés » séparé (100 car., séparés par des virgules,
  sans espaces)** → fourni par langue, en évitant de répéter les mots déjà dans le
  titre/sous-titre (Apple ne compte pas les doublons).
- Portugais : cible **pt-BR (Brésil)** en priorité (marché géant) ; ajoute pt-PT ensuite.
- Vérifie chaque limite après collage (les caractères accentués comptent pour 1).
